# 💰 Контроль бюджета (Трекер бюджета)

Веб-приложение для учёта личных расходов. Простой, быстрый и интуитивно понятный инструмент для контроля бюджета без регистрации и установки.

---

## 📖 Описание

**«Контроль бюджета»** — это клиентское веб-приложение, которое помогает пользователям отслеживать свои расходы, фильтровать их по категориям и видеть общую сумму трат. Проект разработан в рамках индивидуального проекта студентом Московского международного колледжа цифровых технологий «Академия ТОП».

Приложение работает по принципу **SPA (Single Page Application)** — все данные хранятся в оперативной памяти браузера, а интерфейс обновляется динамически без перезагрузки страницы. Это делает его быстрым и удобным для повседневного использования.

Проект создан для людей, которые хотят простой и бесплатный инструмент для контроля личных финансов без необходимости устанавливать приложения или проходить регистрацию.

---

## ⚙️ Функционал

| Функция | Описание |
|---------|----------|
| ➕ **Добавление расходов** | Выбор категории, ввод суммы, даты и описания |
| 📋 **Просмотр списка** | Все расходы отображаются в виде карточек |
| 🔍 **Фильтрация по категориям** | Фильтр: «Питание», «Дорога», «Досуг», «Показать всё» |
| 💰 **Подсчёт общей суммы** | Автоматический пересчёт суммы по текущему фильтру |
| 🛡️ **Защита от XSS** | Экранирование специальных символов |
| 📱 **Адаптивный дизайн** | Корректное отображение на мобильных и десктопных устройствах |
| 🎨 **Светлая тема** | Чистый и читаемый интерфейс |
